# SifreOlusturucu
Windows Forms ile geliştirilmiş güçlü şifre oluşturma uygulaması.

## Özellikler
- Karakter uzunluğu seçme
- Harf, rakam ve sembol seçenekleri
- Rastgele şifre üretimi

## Teknolojiler
- C#
- Windows Forms
- .NET Framework
